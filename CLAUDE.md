# CLAUDE.md

## 项目概述
全球读书日介绍页面，单页展示型网站。

## 技术选型
- 纯 HTML + CSS + JS，无框架依赖
- 响应式设计，移动端友好

## 文件结构
- `world-book-day.html` — 页面主文件

## 设计约定
- 主色：UNESCO 深蓝 `#003da5`，辅以暖橙 `#e8772e` 点缀
- 字体：系统字体栈
- 动画：入场淡入、视差滚动效果

## 内容模块
1. Hero — 大标题 + 今日日期 + 背景图
2. 来历 — 圣乔治屠龙传说
3. 历史 — 1995年 UNESCO 正式确立，背后故事
4. 意义 — 阅读、文化、版权三大价值
5. 呼吁 — CTA 按钮跳转书籍推荐

## 约束
- 单文件，无外部依赖（图片用 Unsplash CDN）
- 无需部署配置文件
